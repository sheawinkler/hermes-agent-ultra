# Hermes Agent Ultra v0.14.2

Release date: 2026-05-26

This patch release keeps the `v0.14.1` systems diagnostics behavior and fixes the CI clippy gate that blocked `main`.

## Highlights

- Refactored `hermes systems ...` CLI dispatch to use a single options struct instead of an eight-argument handler.
- Preserved the MCP, ACP, replay, handoff, provider, release, and provenance diagnostics introduced in `v0.14.1`.
- Bumped the workspace package version to `0.14.2`.

## Verification

Run before tagging:

```bash
cargo fmt --all --check
bash scripts/clippy-warning-gate.sh --check
cargo test -p hermes-cli systems -- --nocapture
cargo test -p hermes-cli auth_store_write_is_atomic_owner_only_and_cleans_tmp -- --nocapture
cargo test -p hermes-acp -p hermes-mcp

python3 scripts/run-security-release-gate-v2.py --repo-root .
```
