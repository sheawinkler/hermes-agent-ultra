# Hermes Agent Ultra v0.14.1

Release date: 2026-05-25

This patch release removes unproven benchmark-superlative product framing from the repo and replaces it with evidence-grounded system integration diagnostics.

## Highlights

- Replaced the former benchmark-superlative command with `hermes systems ...` for implemented system surfaces only.
- Added integrated diagnostics for MCP, ACP, replay trace logs, handoff queues/contracts, provider capabilities, release gates, and provenance wiring.
- Replay diagnostics now inspect existing `logs/replay/*.jsonl` hash-chain metadata instead of writing isolated sample events.
- MCP diagnostics probe the real in-process MCP handler, including resources and prompts.
- ACP diagnostics probe the real ACP handler across initialize, session, prompt, cancel, tools, and status methods.
- Handoff diagnostics inspect the existing gateway handoff queue and preserve the Kanban structured handoff contract.
- Kept OAuth credential persistence hardening from v0.14.0: auth stores now use atomic owner-only writes.
- Removed unproven benchmark-superlative wording and aliases from first-party code, docs, and bundled optional-skill references.

## Verification
